# Aplicativo Be Hero

Treinamento disponibilizado pela RocketSeat, neste módulo foi desenvolvido o aplicativo para o projeto Be Hero.

Neste projeto desenvolvido em node foram utilizadas as seguintes lib, frameworks e módulos:
- React
- ReactNative
- Axios
- Expo
